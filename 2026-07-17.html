月之暗面发布 Kimi K3：全球首个开源3T级模型，综合实力逼近 GPT-5.6 Sol

7月17日凌晨，月之暗面发布 Kimi K3——2.8万亿参数 MoE 架构，基于自研 KDA（Kimi Delta Attention）和注意力残差（AttnRes）构建，原生视觉理解，100万 token 上下文窗口。这是全球首个开源3T级模型。架构效率较 K2 提升约2.5倍，有效激活16/896个专家。评测综合水平仅次于 Claude Fable 5 和 GPT-5.6 Sol。完整权重将于7月27日前开源。年内已完成6轮融资，投前估值315亿美元，ARR 突破3亿美元（API 收入占比超七成）。

月之暗面以接近 GPT-5.6 Sol 和 Claude Fable 5 的实力重返开源牌桌，标志着国产 AI 从"能用"走向"能打"。2.8T 参数 + 100万上下文 + KDA 新架构的组合，让开源模型的"天花板"首次上移到接近闭源旗舰的水准。这一信号叠加 DeepSeek 估值冲刺、WAICO 在上海落地——中国 AI 在同一天完成了"技术发布 + 资本定价 + 制度构建"的三线叙事。

Google Gemini 3.5 Pro 发布延期：编程能力未达预期，市值蒸发2000亿美元

据彭博社7月16日报道，Google 旗舰 AI 模型 Gemini 3.5 Pro 发布已滞后数月，主因编程能力未达内部预期。上月底曾更新训练数据试图补齐短板，但结果令人失望。消息公布后 Google 股价跌4.43%，市值蒸发约2000亿美元。10名现任及前任员工透露，内部挫败感蔓延，部分 AI 研究员已跳槽至 Anthropic。多团队（DeepMind、Cloud、Android）并行开发 AI 编程工具，资源分散加剧了拖延。此前发布的 3.5 Flash 口碑参差不齐，部分用户已转向 Claude Sonnet 5。

这不是 Google 第一次在模型发布时间线上食言——但这次不同。过去 Google 可以靠搜索数据优势和工程规模兜底，如今 Anthropic 在代码和 Agent 场景持续断层领先、OpenAI 在最快迭代节奏中已形成正循环，Gemini 的编程能力短板让它连"综合第一梯队"的定位都在动摇。创始人 Brin 推动布局被内部派系消解、部分工程师仍秉持"代码应由人类编写"原则——一家以系统工程能力自豪的公司，如今最大的敌人可能是自己的组织架构。

小米发布 Robotics-1：10万小时数据预训练，国内首次系统性验证具身智能 Scaling Law

7月16日，小米发布具身基座模型 Xiaomi-Robotics-1，基于10万小时真实世界操作轨迹预训练 + 约1.1万小时跨本体数据后训练。国内首次在机器人策略模型中系统性验证 Scaling Law：预训练数据和模型参数增大时，真机成功率持续稳定提升。四项真实任务上平均成功率75%（每任务 <10h 演示数据），远超 π0.5 的40%。RoboDojo 基准 13.93% 断档第一（第二 8.80%），RoboCasa、VLABench 四榜均第一。小米三天三连发：7/14工厂实习 → 7/15统一生成模型 U0 → 7/16基座模型，完成"本体—数据—模型"闭环。

小米这波密集发布信号清楚：具身智能不再是"能不能动"的炫技，而是"能不能复现 Scaling Law"的系统工程。10万小时真实数据的壁垒不是烧钱能堆出来的——需用自研 UMI 设备逐帧采集、标注、清洗。当别家还在纠结硬件方案，小米已把战火烧到"数据规模驱动能力进化"层面。具身智能的"ChatGPT 时刻"还远，但工业化路径已有人画出图纸。

世界人工智能合作组织（WAICO）成立：29国签署协定，总部设在上海

7月16日，29国代表在上海签署《成立世界人工智能合作组织协定》（WAICO），总部设在上海。王毅代表中国政府签署协定，哈萨克斯坦、老挝、巴基斯坦、俄罗斯、印尼等29国成为创始成员国。联合国秘书长古特雷斯出席仪式。组织定位为独立政府间国际组织，遵循《联合国宪章》宗旨，秉持共商共建共享原则。中国于2025年7月首次提出该倡议。组织仍对后续新成员国家开放，围绕科学研究、技术开发、系统安全、全球治理四大方向构建合作。

在 WAIC 开幕前一天签署 WAICO 协定，时间窗口的精心设计不言自明。29个创始成员国以全球南方国家为主体——这不是 G7 主导的 AI 治理框架，而是另一条多边路线的制度性落地。总部设在上海、古特雷斯站台、王毅亲自签署，这套组合拳表明中国在 AI 全球治理领域已从"议题参与者"升级为"规则搭建者"。

日本 Noetra × NVIDIA：全球首个国家级物理 AI 基础设施，2.75万颗 Rubin GPU

7月16日，NVIDIA 宣布与日本 Noetra 合作建设全球首个国家级物理 AI 基础设施。该 AI 工厂配备13,750颗 Vera CPU 和27,500颗 Rubin GPU（382个 NVL72 机柜），总功率140MW，基于 NVIDIA DSX 平台和 Spectrum-X 网络构建。项目隶属日本 FRONTia 国家计划，由经济产业省支持。Noetra 由索尼、软银、NEC、本田牵头，44家企业和机构参与。首年拨款3873亿日元（约24亿美元），5年总预算最高1万亿日元。计划2027年4月开工，2028年6月投运，目标到2040年占据全球 AI 机器人市场30%以上。

这不是一次普通的 GPU 采购——这是一个主权国家将"AI 基础设施建设"提升到与高速公路、电网同等级别的战略性投入。140MW、2.75万颗下一代 Rubin GPU、2028年投运——时间线和投资规模放在全球都是第一梯队。值得注意的是 Noetra 选择"开源预训练权重+广泛共享"而非封闭自用，这让 FRONTia 具备了公共品属性。日本的真正赌注不在模型本身，而在机器人产业。

NeuronSoup：无反向传播的异步神经网络，遗传算法进化生成，模型仅115KB

7月16日提交 arXiv（2607.15217）。NeuronSoup 提出全新神经网络架构：用异步、时延调制的信号传播替代同步逐层处理，共享神经元池替代独立层。204条路径通过266个隐藏神经元（156个被多路径共享），用遗传算法（14,602个基因）进化而非反向传播训练。10,000代后 MNIST 分类达85.9%，模型仅115KB。信号在共享神经元上产生相长/相消干扰，类似生物神经元的突触交互。

这更像概念验证而非实用突破，但它触碰到的核心问题是真问题：反向传播是人类目前训练神经网络几乎唯一的范式，但它有根本局限——需要可微计算图、固定计算深度、无法自然处理异步信号。NeuronSoup 用进化算法绕开这些，虽然精度不足以实用，但"异步信号干扰"机制在神经科学中有天然对应。如果这个方向能走下去，最有趣的不是替代反向传播，而是催生出真正模拟生物神经元的异构计算硬件。

腾讯混元 HiLS-Attention：分层地标稀疏注意力，512K 上下文推理加速14倍

腾讯混元联合多所高校提出 HiLS-Attention（Hierarchical Landmark Sparse Attention）分层地标稀疏注意力机制。在512K超长上下文场景中，推理速度较全量注意力提升近14倍，长文本检索准确率显著提升。核心思路："不是所有 token 都值得同等计算权重"——通过分层地标策略在保持检索精度的同时大幅压缩注意力计算量。论文已开源。

长上下文的高效推理是当前所有大模型工程化部署中绕不过去的瓶颈。HiLS-Attention 的价值在于把"力大砖飞"的全注意力计算，变成了更有结构效率的稀疏策略。14倍加速在512K上下文这个尺度上，意味着原本需要高端 GPU 集群才能跑的长文档任务，现在单卡也许就能处理——这对 RAG、代码库理解、长文档 QA 等实际场景的部署成本影响不小。

腾讯云开源 CubeSandbox：AI Agent 即时安全沙箱，GitHub 今日665星

腾讯云开源 CubeSandbox，Rust 编写，为 AI Agent 提供即时、并发、安全的轻量级沙箱环境。支持容器化和微 VM 隔离，可实现秒级启动，适用于 Agent 执行代码、运行命令等高风险操作场景。GitHub 今日665星，累计8,437星。与 AWS Firecracker 同源的微 VM 思路，但更轻量、更面向 Agent 场景。

Agent 安全沙箱正在从"nice to have"变成"must have"。随着 Agent 从对话走向实际执行代码/命令/文件操作，沙箱化的需求已经从安全团队的 checklist 变成了 Agent 基础设施的标配。腾讯云在这个赛道的开源投入，某种程度上是在为"Agent 时代的计算基础设施标准"提前占位——谁先定义 Agent 运行环境的安全边界，谁就有可能在下一轮生态竞争中拿到话语权。
