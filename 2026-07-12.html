AgentTether：LLM Agent 运行时故障自修复，59%-65% 成功率

北大 / 微软提出 AgentTether / 将每次 Agent 运行抽象为 Transition Units + Critical Transition Graph / 离线正常模型 + 在线图检测联合定位故障根因 / 261 个 tau-bench 任务评估：Qwen3.7-max 修复率 59%，GPT-5.4 修复率 65% / 无需修改底层 Agent 或环境，即插即用

Agent 落地最大的工程痛点不是"会不会做"，而是"做错了怎么办"。AgentTether 的思路务实：不给 Agent 加新能力，而是在它跑偏时精准定位 + 引导修复。59%-65% 的修复率不算惊艳，但"不改模型，只加一层诊断壳"的路径，比重新训练或 prompt 调参更可规模化。这是 Agent 可靠性工程从论文走向生产的关键尝试。
