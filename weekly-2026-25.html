政府第一次关停前沿模型

Anthropic Fable 5 上线 72 小时被出口管制叫停，SpaceX $750 亿 IPO 收盘 2.1 万亿，AI 监管与资本同时加速。
这一周的三条主线：一是 Anthropic 的 Mythos 级模型发布即遭美国政府出口管制封杀，开创商业 AI 模型被强制下线的先例；二是 SpaceX 以「算力地主」叙事完成史上最大 IPO，Google 和 Anthropic 合计 $700 亿租约构成核心估值逻辑；三是 OpenAI 在 S-1 递交、降价狙击、收购 Ona 三线并进，IPO 前加速补齐商业短板。安全监管、算力基础设施、Agent 商业化——三条线在此交汇。

美国政府首次动用出口管制强制关停前沿 AI 模型
Anthropic 于 6 月 9 日发布首个公开版 Mythos 级模型 Fable 5，72 小时后收到美国商务部出口管制指令，要求立即切断所有外国国民访问权限。因无法在技术层面区分国籍，Anthropic 被迫对全球所有用户停服。导火索为另一家公司声称突破 Mythos 安全防护，但 Anthropic 审查后认定漏洞有限且非通用。Anthropic 公开表示该决定建立在"严重误解"之上，警告若此标准推行将实质叫停所有前沿模型部署。

技 术 演 进 趋 势
扩散模型挑战自回归：文本生成新范式从论文走向工程
Google DeepMind 发布 DiffusionGemma（26B MoE / 3.8B 激活），每次前向传播并行生成 256 tokens，单 H100 上 1000+ tok/s、RTX 5090 上 700+ tok/s，Apache 2.0 开源。相比自回归逐 token 串行生成，扩散范式在推理速度上有结构性优势——代价是双向注意力机制需要更多显存（18GB）。这不是第一个文本扩散模型，但它是第一个在工程可用性上接近自回归方案的产品级实现。同周 arXiv 上 HRM-Text 用 1B 参数跑出 7B 水平、TwinQuant 把 4-bit 量化做到近 FP16 精度——推理效率优化的方向正在分化：一条路压自回归的 KV cache（ReasonAlloc），另一条路换生成范式。
依据：DiffusionGemma 发布（6/11）+ HRM-Text 论文（6/9）+ TwinQuant 论文（6/1）+ ReasonAlloc KV cache 分配（6/10）
Agent 安全从"事后审计"转向"实时门控+供应链扫描"
本周密集出现三类 Agent 安全工具：OpenAI 发布 ChatGPT Lockdown Mode（一键关闭 Agent Mode/实时浏览/Deep Research），NVIDIA 开源 SkillSpector（静态+LLM 语义扫描 Agent 技能漏洞，64 漏洞 16 类别，0-100 评分），GitHub 上出现 agent-gate（MCP 质量门控服务器，SHA256 链式收据 + fail-closed 检查）。加上 73 个微软开源包被植入窃密代码专门针对 AI Agent——安全防御正从"模型对齐"向"Agent 执行链全链路管控"迁移。这不仅是技术选择，也是监管压力下的必然：Anthropic Fable 5 被出口管制关停的核心争议，正是安全边界如何定义。
依据：ChatGPT Lockdown Mode（6/7）+ NVIDIA SkillSpector（6/12）+ agent-gate（6/8）+ 微软供应链攻击（6/8）+ Anthropic 出口管制事件（6/12）
编程 Agent 从"云端 IDE 插件"走向"终端持久化 + 多模型混编"
小米开源 MiMo Code V0.1（持久记忆三重机制 + Compose 模式 1→12 协同 + 支持 DeepSeek/Kimi/GLM），DeepSeek 发布 Reasonix（原生编程 Agent + 前缀缓存稳定性设计），Cohere 开源 North Mini Code（30B MoE / 单 H100 可运行），OpenAI 收购 Ona 使 Codex Agent 在客户私有云持续运行。共同信号：编程 Agent 不再是浏览器里的代码补全工具，而是独立进程级的开发者助手，需要持久记忆、多模型路由、本地/私有云执行环境。小米和 DeepSeek 的加入说明中国厂商开始从"模型层"向"Agent 基础设施层"渗透。
依据：MiMo Code V0.1 开源（6/11）+ DeepSeek-Reasonix（6/10）+ Cohere North Mini Code（6/10）+ OpenAI 收购 Ona（6/11）
Agent 成本优化成为独立赛道，Token 降本工具链成熟
OpenSquilla 用微内核架构 + 智能模型路由，25 个混合任务成本从 $6.2 降至 $0.68（降幅 89%）。SkillAxe 让 LLM 自主打磨 Agent 技能，pass rate 提升 28%，弥合人类差距从 47% 缩至 67%。这不再是"换个更便宜的模型"的粗粒度优化，而是从路由策略、技能复用、执行预算分配等多个维度精细拆解 Agent 的 Token 消耗结构。当企业级 Agent 从 PoC 走向生产，成本治理将成为和功能开发同等重要的工程命题。
依据：OpenSquilla 微内核 Agent（6/11）+ SkillAxe 无监督技能打磨（6/10）+ Google AI Plus 降价（6/10）+ OpenAI 降价传闻（6/11）
商 业 格 局 变 化
AI 算力从"自建"走向"租赁"，SpaceX 成为最大第三方算力供应商
SpaceX 招股书披露：Anthropic 月付 $12.5 亿租用孟菲斯 Colossus 1/2，Google 月付 $9.2 亿租用 11 万张 NVIDIA GPU，两份合同到 2029 年合计超 $700 亿。SpaceX 自己做 AI（xAI）年亏 $64 亿，出租算力年入 $260 亿。这个"挖金不如卖铲"的逻辑，让 SpaceX 在 IPO 估值中把 AI 基础设施叙事填进了 Starlink 和火箭发射之间的万亿美元缺口。但风险也很明确：Anthropic 合同任一方 90 天可终止，算力租赁不是长期锁定。
AI 安全监管从"自愿框架"升级为"强制干预"，出口管制成为新工具
美国商务部首次动用出口管制权力叫停已部署的商业 AI 模型，开创先例。此前 AI 监管主要依赖企业自愿承诺和行政命令，此次直接以国家安全为由禁运。Anthropic 自己也在同周向华盛顿提交了 AI 监管手册（10^25 FLOPs 阈值 + $5 亿 AI 收入门槛），但政府显然选择了更激进的路径。德国法院同周裁定 Google 对 AI 生成内容负法律责任。监管层面对 AI 的约束正在从"软法"转为"硬法"，从"事后追责"转为"事前禁运"。
AI IPO 赛跑：三家巨头同周推进，但监管逆风同步加码
SpaceX 已上市（2.1 万亿收盘），Anthropic S-1 已递交（$9650 亿估值），OpenAI S-1 秘密递交（$8500 亿）。三家公司合计 IPO 规模逼近 $2 万亿。但同周 OpenAI 遭多州 AG 调查（广告政策 + 健康数据），Anthropic 被出口管制打击，xAI 前工程师起诉安全风险被解雇。资本窗口和监管窗口同时打开，IPO 不是终点，而是监管博弈的起点。
Agent 支付基础设施落地：Visa 接入 ChatGPT，资金流闭环成型
Visa 支付网络嵌入 ChatGPT，Agent 首次可以独立完成从选购到付款的全流程。这不是"AI 推荐你买什么"，而是"AI 替你花出去"。消费上限和商户限制提供安全阀，但核心变化是 Agent 从信息获取工具变为交易执行者。微信 AI 开放小程序接口（京东首批内测）、阿里千问发布高考志愿 Agent——中国市场的 Agent 落地更偏服务和内容场景，而美国市场的 Agent 正在向金融和交易渗透。
中国 AI 厂商从"模型发布"转向"组织重构+Agent 落地"
阿里成立 Token Foundry 事业部、CEO 直管 AI 全链路、周靖人转首席科学家，这是组织层面的 AI 优先战略升级。小米开源 MiMo Code 编程 Agent。云知声发布 2660 亿参数 U2 大模型。智谱联合清华开源 SCAIL-2 动画生成。DeepSeek 首轮融资 500 亿、昇腾 910C 完成万亿参数全参数后训练。信号明确：中国厂商的竞争重心正在从"谁的模型跑分更高"转向"谁的 Agent 先落地场景+谁的算力自主可控"。
