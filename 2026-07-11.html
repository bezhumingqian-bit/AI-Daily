苹果起诉 OpenAI 窃取商业机密：硅谷最彻底的"友尽"

7月10日，苹果向加州北区联邦法院提起诉讼，指控 OpenAI 系统性窃取 iPhone 商业机密用于开发消费级 AI 硬件。诉状点名 OpenAI 首席硬件官 Tang Tan（前苹果产品设计副总裁，任职24年）及前 iPhone 工程师，指控其在面试中要求应聘者携带苹果"实际零件"和"CAD图稿"。苹果称已有400余名前员工入职 OpenAI，要求销毁所有侵权材料并重新设计即将推出的产品。

GPT-5.6 Sol Ultra 调 64 个 Agent，1 小时证明 50 年数学猜想

OpenAI宣布GPT-5.6 Sol Ultra调用64个子Agent，用时不到1小时，为1976年提出的"循环双覆盖猜想"给出完整证明。OpenAI已公开完整prompt和证明全文，并开源Lean形式化版本——相关代码已通过Lean内核检查，意味着证明在形式化意义上经得起机器验证。不过数学界尚未完成独立审查。

GLM-5.2 杀进硅谷：Databricks 弃 Opus 换国产开源，省 34%

Databricks联合创始人Matei Zaharia 7月8日公布内部基准测试：GLM-5.2（753B参数MoE，MIT协议）与Opus 4.8在编程任务上统计持平（通过率82-90%），但单任务成本$1.28 vs $1.94——省34%。Coinbase将模型切换至GLM-5.2+Kimi 2.7组合后AI成本腰斩。OpenRouter平台数据显示中国模型周流量占比从11%升至30%以上，定价比西方模型低六到九成。

今天的GLM-5.2和昨天的GPT-5.6是同一片星空下的两个故事：一边是前沿模型在"能做多难的事"上拉高天花板，另一边是开源模型在"多便宜能做到"上凿穿底价。Databricks不是一家边缘创业公司——它是数据基础设施的核心玩家，它"叛逃"Opus是硅谷用钱包投票的信号。

Claude Code 内置浏览器上线：Agent 直接上网查文档改 Bug

7月11日，Anthropic为Claude Code桌面端加入内置浏览器面板（Cmd+Shift+B调出），Claude可直接打开技术文档、GitHub issue、设计稿等网站，读取、点击、填表，无需在编辑器和Chrome之间切屏。浏览器使用独立配置——不与个人登录状态和历史共享，首次操作网站需用户授权（允许一次/始终允许/拒绝）。购买、注册和验证码仍需用户亲手完成。

开发者的体验会变成"告诉Claude去读那个issue，然后继续写代码"而不是"打开Chrome→复制链接→粘贴到Claude→切回来"。这听着像一个small feature，实际是Agent从"代码生成器"到"开发工作台"的关键一步——agent需要的不只是推理能力，还有在真实互联网上主动获取信息的能力。

MCP 史上最大修订：2026-07-28 RC 发布，协议全面无状态化

MCP 协议发布自推出以来最大规模修订的 Release Candidate。核心变化：① 移除 initialize 握手和 Mcp-Session-Id，请求自包含化——任何MCP请求可落在任何服务器实例上，不再需要 sticky session；② 新增 Extensions 框架、Tasks 扩展、MCP Apps（服务端渲染UI）；③ 废弃 Roots/Sampling/Logging 三个旧特性，进入12个月迁移倒计时；④ 路由改为 Mcp-Method 头（无需解析JSON body），Tool schema 升级至 JSON Schema 2020-12。最终规范7月28日锁定，6个SEP共同实现。Python v2 / TypeScript v2 / Go / C# SDK Beta已同步发布。

从"有状态的聊天协议"到"无状态的HTTP服务协议"——MCP的这次跃迁本质上是把自己从玩具级基础设施升级为可水平扩展的生产级基础设施。移除session带来的工程简化很具体：之前需要sticky routing+共享session store的部署，现在一个nginx轮询就够了。MCP Apps的加入则意味着MCP正在从"后端工具接口"进化成"全栈应用平台"。
